 ## 1.字体文件 字体使用CDN
 > 字体为网络搜集，如侵犯版权请联系我删除。
### 使用方法
```
@font-face {
	font-family: "Flama";
	font-weight: normal;
	font-style: normal;
	src: url('https://cdn.jsdelivr.net/gh/TAOYONG12138/font/文悦古典明朝.otf') format('opentype');
}
```
>otf格式：format('opentype')   
ttf格式：format('truetype')
